# Step19: RDS (Relational Database Service)

## What is RDS?
RDS is a managed database service in AWS.
It is used to store and manage structured data.

---
## Structured Data
Structured data is organized in tables with defined schemas.

Example:
- User ID
- Name
- Email

---
## Managed Database
RDS handles operational tasks such as:
- Backup
- Patching
- Failure recovery

This allows developers to focus on data and application logic.

---
## Difference from S3
- RDS: query-based access (SQL)
- S3: key-based access (object storage)

RDS is used for managing structured data, while S3 is used for storing files.

---
## Key Concept
RDS = managed service for structured data

---

## Summary

RDS provides a scalable and managed way to store and query structured data in AWS.
