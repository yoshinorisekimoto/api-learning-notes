# Step16: EC2 (Elastic Compute Cloud)

## What is EC2?
EC2 is a virtual server in AWS.
It allows you to run applications on a cloud-based computer.

---
## What is a Virtual Machine?
A virtual machine (VM) is a software-based computer that runs on physical hardware.
EC2 provides virtual machines that can be created and managed on demand.

---
## Why EC2 is Needed
Traditional servers require physical setup and maintenance.
EC2 solves this by providing:
- On-demand provisioning
- Flexible scaling
- No need to manage physical hardware

---
## Role of EC2
EC2 is used to:
- Run applications
- Execute backend logic
- Process API requests

---
## Relationship to VPC
EC2 instances are deployed within a VPC:
- Public Subnet: accessible from the internet
- Private Subnet: internal use only

---
## Key Concept
EC2 = Virtual server for executing business logic

---

## Summary

EC2 provides scalable and flexible virtual servers that allow applications to run in the cloud without managing physical infrastructure.
