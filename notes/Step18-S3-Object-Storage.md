# Step18: S3 (Object Storage)

## What is S3?
S3 is an object storage service in AWS.
It is used to store files such as images, videos, and documents.

---
## Object Storage
Object storage is a way to store data as files (objects).
Examples include:
- Images
- PDFs
- Videos
- Logs

---
## Path (Key-Based Access)
In S3, each file is identified by a key (path-like string), such as:
images/profile/user1.jpg

---

## Difference from Database
- S3 retrieves data using a key (path)
- Databases retrieve data using queries (SQL)
S3 is used for storing files, while databases are used for managing structured data.

---

## Key Concept
S3 = file storage using key-based (path-like) access

---
## Summary
S3 provides scalable storage for unstructured data, where files are accessed using a key instead of queries.
