# Step17: Lambda (Serverless)

## What is Lambda?
Lambda is a serverless compute service in AWS.
It runs code only when triggered by an event.

---
## Key Difference from EC2
- EC2: always running server
- Lambda: runs only when needed

Lambda removes the need to manage servers.

---
## Event-Driven Model
Lambda functions are triggered by events such as:
- API requests
- File uploads
- System events

---
## Scaling
Lambda automatically scales based on incoming requests.
No manual scaling is required.

---
## Key Concept
Lambda = event-driven execution without server management

---
## Summary
Lambda provides a serverless way to run code, improving efficiency and scalability compared to EC2.
